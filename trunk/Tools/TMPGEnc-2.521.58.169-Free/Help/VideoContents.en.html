[Project wizard] - [select source file] - [Content of video]

At [Content of video], specifies information of source video
By inputting appropreate value, TMPGEnc encodes in most optimized method.

Video movie

The source consists only from video movie(29.97fps progressive/interlaced)

Select this option if you are trying to encode movies captured from digital camera.
Film movie The source consists only from film movie(substantially 24fps).
This mode can be used for 29.97fps interlaced file movie.

Select this option if you are trying to encode film / anime movies.
Video/Film mixed Select this option if you have mixed source.

If your source is none of the above, select "Video movie".
