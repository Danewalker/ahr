			======================================
			               TMPGEnc 2.5
			======================================

１．初めに

　これは AVI 形式の動画を MPEG-1/2 に変換するソフトウェアです。

２．TMPGEnc を利用するにあたって

　推奨環境は以下の通りです。

　・PentiumIII 500MHz 以上
　・64MB 以上のメモリ
　・100MB 以上の空きを持ったハードディスク

　利用するには以下が必要になります。

　・Windows 98/98SE/Me/NT4.0/2000/XP HomeEdition/XP Professional のいずれか
　・800x600 16ビットカラー以上の環境

　下記の条件の環境では、さらに快適に動作します。

　・画面解像度 1152x864 24ビットカラー 以上
　・MMXテクノロジー 対応 CPU
　・Streaming SIMD Extensions 対応 CPU (Pentium III など)
　・Streaming SIMD Extensions2 対応 CPU (Pentium 4 など)
  ・Enhanced 3D Now! 対応 CPU (Athlon など)
　・3D Now! 対応 CPU(ただし速度アップ効果はわずかです)
　・マルチプロセッサ(Windows NT4.0/2000/XP Professional環境において)
　・DirectX 6 以降のランタイム
　・DirectShow ランタイム
