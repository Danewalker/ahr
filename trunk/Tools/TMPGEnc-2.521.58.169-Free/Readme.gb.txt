			======================================
			              TMPGEnc 2.5
			======================================

1. 软件

 用于压缩AVI(*.AVI)文件成MPEG-1/2.

2. 设备

  建议最小配置:

  - PentiumIII 500MHz或更高
  - RAM : 64MB 或更高
  - 至少100MB硬盘空间


  以及如下配置：

  - 操作系统: Windows 98/98SE/Me/NT4.0/2000/XP HomeEdition/XP Professional/
  - 显示器: 800x600 / 16bit color or higher

  本软件建议配置 

  - 分辨率为1152 x 864 24bit彩色以上 
  - 支持 MMX 技术的中央处理器 
  - 支持Streaming SIMD Extensions的中央处理器(如：PentiumIII) 
  - 支持扩展Enhanced 3D Now!的中央处理器(如：Athlon) 
  - 支持3D Now!的中央处理器(可以支持瞬间加速) 
  - 一个以上CPU (Windows NT/2000硬件环境) 
  - 支持Streaming SIMD Extensions 2的中央处理器(Pentium4) 
  - DirectX 6 或以上
